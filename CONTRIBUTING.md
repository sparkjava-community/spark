## Getting Started

1. Fork this repository
2. Create a feature branch: `git checkout -b fix/issue-123`
3. Make your changes
4. Run tests: `mvn clean test`
5. Commit: `git commit -m "Fix: description"`
6. Push: `git push origin fix/issue-123`
7. Open a Pull Request
